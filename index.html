CSS Box Model Showcase

About the Box Model

Every HTML element is a box. This box consists of content, padding, border, and margin. Understanding these components is key to building good layouts.

Applying Padding

This element has padding. Padding adds space inside the border, pushing the content away from the edges.

Applying Margin

This element has a margin. Margin adds space outside the border, creating distance between elements.

Combining Properties

This box demonstrates all components: content, padding, a border, and a margin. The background color helps visualize the content area.
